Para criar um site, primeiro temos que abrir o VS Code, criar um arquivo com nome "index.Html" e digitar "!' e apertar enter, nisso o código que faz o site funcionar será criado.

Com o código "head" feito você deverá escrever seu código entre os marcadores "Body"

Primeiros marcadores:

O primeiro marcador será o de título, o "h" com <> antes e após,você deverá escrever após ele, caso queira fazer outro título, é só adicionar um ao número após o "h", o segundo será o "

" de parágrafo.
